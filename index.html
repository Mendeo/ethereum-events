Ethereum events

Отслеживайте события, которые генерирует заданный смарт-контракт в сети Ethereum!

Используя расширение для браузера MetaMask и данное веб приложение, вы сможете отслеживать события, которые генерирует любой смарт-контракт в сети Ethereum.
Особенно это веб приложение будет полезно, если вы разработчик смарт-контрактов. В этом случае вы можете с помощью MetaMask подключиться к тестовой сети и отслеживать события от разрабатываемых смарт-контрактов.

Возможности

  • События можно фильтровать по их именам.
  • Отображается дата и время, когда было получено событие, его имя, имена и значения его параметров.

Как пользоваться

  1. Нажмите на кнопку "Подключиться с MetaMask", тогда страница запросит соединение с блокчейном через MetaMask.
  2. После успешного соединения нужно ввести адрес смарт-контракта и его ABI.
  3. Программа начнёт отслеживать события этого контракта после нажатия на кнопку "Начать слушать".
  4. Выбирайте только нужные вам события при помощи фильтра.
  • Веб приложение работает только в вашем браузере через MetaMask. Ничего, кроме языковых настроек вашего браузера, на сервер приложения не передаётся. Но есть одно исключение: если вы решите пожертвовать эфир разработчику, то на сервер отправится ваше сообщение, ваш ethereum адрес и сумма пожертвования.
  • Вы можете посмотреть исходный код на GitHub: https://github.com/Mendeo/ethereum-events

Для работы этого сайта необходимо, чтобы был включён JavaScript в вашем браузере.
