Ethereum events

Watch events generated by the specified smart contract in the Ethereum blockchain network!

Using the MetaMask browser extension and this web application, you can watch events generated by any smart contract on the Ethereum network.
This web application is especially useful if you are a developer of smart contracts. In this case, you can use MetaMask to connect to the test network and watch events from smart contracts that you are developing.

Scope

  • Events can be filtered by their names.
  • Displays event time, event name and names and values of event parameters.

How to use

  1. Click on the "", then the page will request a connection to the blockchain via MetaMask.
  2. After a successful connection, you have to enter the smart contract address and ABI.
  3. The application will start listening the events of this contract after clicking the "" button.
  4. Use the filter to select only the events you need.
  • The web app only works in your browser via MetaMask. Nothing is sent to the application server except the language settings of your browser. But there is one exception: if you decide to donate ether to the developer, your message, your ethereum address and the amount of the donation will be sent to the server.
  • You can view the source code on GitHub: https://github.com/Mendeo/eth-events
