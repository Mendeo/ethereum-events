Отслеживайте события, которые генерирует заданный смарт контракт в сети Ethereum.

Нажмите на кнопку "", тогда страница запросит соединение с блокчейном через MetaMask.
После успешного соединения нужно ввести адрес смарт контракта и его ABI.
Программа начнёт отслеживать события этого контракта после нажатия на кнопку "".

  • Веб приложение работает только в вашем браузере через MetaMask. На сервер приложения не отправляется никаких данных, кроме языковых настроек браузера.
  • Для работы должно быть установлено расширение MetaMask.
  • Вы можете посмотреть исходный код на Github: https://github.com/Mendeo/eth-events
